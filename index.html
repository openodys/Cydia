Welcome to the Odys Group's Cydia Repo.

On this repo we will put different kinds of software we didn't release on the public repos. You can also find beta software we are working on in the repo. If you install these tweaks please make sure to let the developers know about any bugs you find. You can download these beta on your own risk but this is not software for every normal user.
